A simple Elementor addon for rotating elements and containers visually.

== Description ==

**[Elementor Rotate](https://wpbuilt.co/tutorials/how-to-rotate-using-a-custom-elementor-plugin/)** is a straight forward elementor addon that adds a new property on Elementor widgets which allows you to rotate them using a slider.

Features:

*   Easy to use
*   Elementor compatible
*   Open-source
*   Responsive

To manually install this plugin just upload plugin zip file on Plugins>Add New and then that's it!
